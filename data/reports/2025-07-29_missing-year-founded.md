## Year Founded Research Methodology

### Sources for Year Founded Information

1. **Company Website**: Check "About", "History", or "Our Story" pages
2. **LinkedIn**: Company page typically includes founding year
3. **Crunchbase**: Provides detailed company timeline information
4. **BC Business Registry**: Provides incorporation date
5. **Press Releases**: First press release or company announcement
6. **Industry News**: Articles about company founding or launch

### Data Format Guidelines

- Enter Year Founded as a 4-digit number (e.g., `2015`)
- For companies with multiple founding dates (e.g., original founding and incorporation), use the earliest date
- If only approximate year is known, still enter the year
- If month/day is known, still only enter the year in the Year Founded field

### Batch Update Script

After researching year founded information, a batch update script can be created to update multiple organizations at once:

```javascript
// Example batch update script structure
const updates = [
  { id: "page_id_1", yearFounded: 2018 },
  { id: "page_id_2", yearFounded: 2015 },
  // etc.
];

for (const update of updates) {
  await notion.pages.update({
    page_id: update.id,
    properties: {
      "Year Founded": { number: update.yearFounded }
    }
  });
}
```
