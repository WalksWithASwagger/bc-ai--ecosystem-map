## 📋 **DATA QUALITY STANDARDS**

### **ACCEPTANCE CRITERIA:**
✅ **Source Attribution**: Every data point must have clear source citation  
✅ **Date Verification**: All data must have collection/verification timestamps  
✅ **Multi-Source Confirmation**: Critical data requires 2+ source verification  
✅ **Official Priority**: Government/company official sources take precedence  
✅ **Confidence Scoring**: All data tagged with confidence level (50-99%)  
✅ **Update Tracking**: Clear documentation of what was added/changed  

### **REJECTION CRITERIA:**
❌ **No Source**: Data without clear source attribution  
❌ **Single Source Claims**: Unverified single-source data for critical fields  
❌ **Outdated Information**: Data older than 2 years without recent confirmation  
❌ **Low Confidence**: Data with confidence below 70%  
❌ **Duplicate/Conflicting**: Data that contradicts existing verified information
