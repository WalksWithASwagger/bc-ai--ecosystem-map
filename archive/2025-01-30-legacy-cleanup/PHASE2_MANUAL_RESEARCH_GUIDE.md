# Phase 2 Manual Research Guide

*A comprehensive guide for gathering high-quality intelligence on BC AI organizations*

## Overview

This guide provides structured approaches for manually researching organizations to gather verified, high-quality data including LinkedIn profiles, funding information, key personnel, and company intelligence.

## Research Priorities

### 🎯 High-Value Targets (Research First)
1. **Well-funded startups** (likely to have rich LinkedIn profiles)
2. **AI Companies category** organizations
3. **Scale-ups (51-250 employees)**
4. **Organizations with websites but no LinkedIn**
5. **Government-funded initiatives**

### 📊 Data Points to Collect

For each organization, gather:

1. **LinkedIn Company Profile**
   - URL (must be company page, not personal profile)
   - Employee count
   - Company description
   - Founded year
   - Specialties/focus areas
   - Headquarters location

2. **Key People**
   - Founder(s) name and LinkedIn
   - CEO/President name and LinkedIn
   - CTO/VP Engineering name and LinkedIn
   - Head of AI/ML (if applicable)

3. **Funding Information**
   - Total funding raised
   - Latest round (amount, date, stage)
   - Key investors
   - Government grants received

4. **Company Intelligence**
   - Current employee count
   - Year founded
   - Office locations
   - Tech stack/AI focus
   - Key customers/partners

5. **Social Media**
   - Twitter/X handle
   - GitHub organization
   - YouTube channel
   - Blog/Medium

## Research Sources

### Primary Sources (Most Reliable)

1. **LinkedIn**
   - Search: `[Company Name] British Columbia`
   - Verify: Check location, employees, activity
   - Extract: All available data points

2. **Company Websites**
   - Check: /about, /team, /leadership, /news
   - Press releases for funding/partnership info
   - Job postings for tech stack and growth signals

3. **Crunchbase**
   - Funding history
   - Key people
   - Company metrics
   - Investor information

4. **Government Databases**
   - [BC Registry](https://www.bcregistry.gov.bc.ca/) - incorporation dates
   - [Innovation Canada](https://innovation.ised-isde.canada.ca/) - grants/funding
   - [NRC-IRAP](https://nrc.canada.ca/en/support-technology-innovation) - funding recipients

### Secondary Sources

5. **News & Media**
   - [Techcouver](https://techcouver.com/)
   - [BetaKit](https://betakit.com/) - search "British Columbia"
   - [Georgia Straight Tech](https://www.straight.com/tech)
   - Local newspapers tech sections

6. **Industry Directories**
   - [Angel List](https://angel.co/) - startup profiles
   - [Clutch.co](https://clutch.co/) - B2B companies
   - [G2](https://www.g2.com/) - software companies

7. **Social Media**
   - Twitter/X - official accounts, announcements
   - GitHub - organization profiles, activity
   - YouTube - company channels, demos

## Search Techniques

### LinkedIn Search Strategies

1. **Direct Search**
   ```
   site:linkedin.com/company "[Company Name]"
   ```

2. **Location-based**
   ```
   "[Company Name]" "Vancouver" OR "British Columbia" site:linkedin.com
   ```

3. **Industry-specific**
   ```
   "[Company Name]" "artificial intelligence" OR "machine learning" site:linkedin.com
   ```

### Funding Research

1. **News Search**
   ```
   "[Company Name]" "raised" OR "funding" OR "investment" "Vancouver"
   ```

2. **Press Release Search**
   ```
   "[Company Name]" "Series A" OR "Series B" OR "seed funding" site:newswire.ca
   ```

### People Research

1. **Leadership Search**
   ```
   "[Company Name]" "CEO" OR "founder" OR "CTO" LinkedIn
   ```

2. **Team Pages**
   ```
   site:[company-website.com] "team" OR "about us" OR "leadership"
   ```

## Data Validation Checklist

Before adding any data, verify:

- [ ] LinkedIn URL is a company page (not personal profile)
- [ ] LinkedIn company name matches our database entry
- [ ] Location confirms BC/Vancouver presence
- [ ] Funding information has a date (within last 5 years)
- [ ] Key people are current (check their LinkedIn)
- [ ] Employee count is recent (LinkedIn shows "employees on LinkedIn")
- [ ] Website is currently active and matches company

## Research Workflow

### Step 1: Preparation (5 min)
1. Open organization in Notion
2. Note existing data (website, category)
3. Open research tracking spreadsheet

### Step 2: LinkedIn Research (10 min)
1. Search LinkedIn for company
2. Verify it's the correct company
3. Extract all available data points
4. Check employee profiles for additional intel

### Step 3: Web Research (10 min)
1. Visit company website
2. Check About/Team pages
3. Search for recent news/funding
4. Look for social media links

### Step 4: Deep Dive (15 min)
1. Search Crunchbase (if available)
2. Google News search for recent updates
3. Check government databases
4. Industry publication search

### Step 5: Documentation (5 min)
1. Record all findings with sources
2. Note confidence level for each data point
3. Flag any uncertainties
4. Update tracking spreadsheet

## Quality Standards

### ✅ High-Quality Data
- Recent (within last year)
- From official sources
- Cross-verified from multiple sources
- Complete with context

### ❌ Avoid
- Outdated information (>2 years old)
- Unverified claims
- Personal LinkedIn profiles as company pages
- Estimates without sources
- AI-generated or guessed data

## Batch Processing Tips

### Efficient Research Order
1. Start with companies that have websites
2. Group by category (e.g., all "AI Companies" together)
3. Use browser tabs efficiently (LinkedIn, Crunchbase, Google)
4. Track progress in spreadsheet

### Time Management
- Allocate 30-45 minutes per organization for deep research
- 10-15 minutes for basic LinkedIn + website check
- Set daily targets (e.g., 10 deep, 20 basic)
- Take breaks every hour

### Collaboration
- Share findings in daily reports
- Flag high-value discoveries
- Note patterns (e.g., "Many biotech startups have Crunchbase profiles")
- Create shared bookmark folders

## Common Challenges & Solutions

### Challenge: No LinkedIn Found
**Solution**: 
- Try variations of company name
- Check if acquired/rebranded
- Look for employee profiles mentioning company
- May genuinely not have LinkedIn (note this)

### Challenge: Multiple LinkedIn Pages
**Solution**:
- Check employee count (higher = likely main page)
- Verify location matches
- Check last activity date
- Look for "official" in description

### Challenge: Outdated Information
**Solution**:
- Note the date of information
- Flag as "historical data"
- Search for recent updates
- Check if company pivoted/shut down

### Challenge: Limited Public Info
**Solution**:
- Check industry associations they might belong to
- Look for conference speaker bios
- Government grant databases
- Patent filings
